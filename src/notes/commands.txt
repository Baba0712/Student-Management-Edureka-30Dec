npm commands
1. npm update <package_name>@latest --to take the latest version
2. npm update <package_name>@version --to take the specific version
3. npm install -- to install the dependncies again.

Angular CLI Commands:
1. ng new <App_name> --to create a new Angular Project
2. npm start/ng serve -- to run our App
3. ng g c <component> -- c is for component , g is for generate
4. ng g service <service> -- to create a class based service.
5. ng g module <module_name> -- to create an angular module
6. ng g guard <guard_name> -- creates a new guard
7. ng g pipe <pipe_name> -- create a new pipe
8. ng g directive <directive_name> --creates a new directive
9. npm run build -- this is to create a development build
10. ng build --prod  : creates production build.
11. npm run test -- to run unit test cases
12. npm run e2e -- to run automation test cases
13: npm run lint -- to display all linting errors
14. ng eject -- gives the webpack config file to us. 
15. npm install @angular/animations@latest @angular/common@latest @angular/compiler@latest @angular/core@latest @angular/forms@latest @angular/http@latest @angular/platform-browser@latest @angular/platform-browser-dynamic@latest @angular/router@latest core-js@latest rxjs@latest zone.js@latest --save
To upgrade the angular to latest version.
